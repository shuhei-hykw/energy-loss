# Project Rules

## Language

- Always respond to the user in Japanese.

## Git

- Do NOT add `Co-Authored-By: Claude` or any AI co-author trailer to
  commit messages.

## Code style

- Use **2 spaces** for indentation in ALL code (Python included).
  Do NOT use 4 spaces or tabs.
- This overrides PEP 8's 4-space default for this project.
- Configure ruff / formatters accordingly.
